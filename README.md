# 🩺 BMI Calculator (Rust CLI)

A simple, fast, and lightweight Body Mass Index (BMI) calculator written in **Rust**. This project is part of my journey to master systems programming and safe memory management.

## ✨ Features
- **Precise Calculation:** Uses Rust's strict typing for accurate results.
- **Instant Feedback:** Categorizes your BMI (Underweight, Normal, Overweight, etc.).
- **Minimalist Design:** Clean Command Line Interface (CLI).

## 🛠 Tech Stack
- **Language:** Rust 🦀
- **UI Framework:** Slint (declarative GUI)
- **Concept:** Reactive UI, control flow, and safe memory management.

## 🚀 How to Run
Make sure you have [Rust](https://rust-lang.org) installed.

1. Clone the repository:
```bash 
git clone https://github.com/Don-Akoppo/bmi-calculator-rust.git
```

2. Run the project:
```bash 
cargo run
```
